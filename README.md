# Home Lab

A personal IT homelab built using VMware workstation on a custom built pc. Built to learn networking, system administration, and IT infrastructure through hands-on experimentation and troubleshooting.

## Environment
-Host: Custom built desktop (Ryzen 7 7800x3D, 32GB Ram)
-Hypervisor: VMware Workstation
-VMs used: Ubuntu Desktop, Kali Linux

## What I've Done So Far
-Deployed and configured nginx web server
-Configured UFW firewll rules and tested with nmap
-Set up SSH, tested authentication logging
-Simulated basic incident response by terminating an active SSH session
-Diagnosed and resolved netowrk issues on both Ubuntu and Kali Linux VMs
